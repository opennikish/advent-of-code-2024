### ROBO WAREHOUSE

This program will clean warehouse based on warehouse map and robo movements.
Wirehouse map will be twice that provided. For example:

```
#######
#...#.#
#.....#
#..OO@#
#..O..#
#.....#
#######

<vv<<^^<<^^
```

Will result into this map:
```
##############
##......##..##
##..........##
##....[][]@.##
##....[]....##
##..........##
##############
```

Run:
```
go run 2.go demo.txt

ROBO WAREHOUSE

##........[]......##
####################
```
